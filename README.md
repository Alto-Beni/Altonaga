# Clash Royale
## ¿Qué es?
Clash Royale es un juego de estrategia en tiempo real, se basa en un enfrentamiento 1v1 o 2v2 de dos bandos, el del rey azul y el del rey rojo, cada jugador tiene un mazo de 8 cartas y una barra de elixir de 0 a 10, hay 120 cartas actualmente, aunque progresivamente se van modificando y agregando nuevas cartas.
## ¿Cómo jugar y armar tu mazo?
tenés que derrotar al rey contrario, destruyendo sus tres torres o destruyendo más de las que te destruyó él a vos antes de que acabe el tiempo(tres minutos estándar, y si después de este tiempo siguen en empate, se agregan 2 minutos de muerte súbita, que implica que el primero en ganar una torre del rival gana, y si aún así después de este tiempo siguen empatados, el juego le quita toda la vida a las torres de los dos y el primero en perder una torre pierde)
