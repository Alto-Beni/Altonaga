# Clash Royale
## ¿Qué es?
Clash Royale es un juego de estrategia en tiempo real, se basa en un enfrentamiento 1v1 o 2v2 de dos bandos, el del rey azul y el del rey rojo, cada jugador tiene un mazo de 8 cartas y una barra de elixir de 0 a 10, hay 120 cartas actualmente, aunque progresivamente se van modificando y agregando nuevas cartas.
## ¿Cómo jugar y armar tu mazo?
